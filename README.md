🔺 Classificador de Triângulos
Classificador de Triângulos 🔺 — Programa em Python que recebe as medidas dos três lados de um triângulo e retorna se ele é equilátero, isósceles ou escaleno. Simples, direto e ótimo para iniciantes em lógica de programação!

Informe os três lados e descubra qual é o seu triângulo!

📐 Tipos de Triângulo:
🟢Equilátero os três lados são iguais
🟡Isósceles dois lados são iguais
🔴Escaleno todos os lados são diferentes

💻 Exemplo de uso
dê a medida do lado 1 do seu triângulo: 3
dê a medida do lado 2 do seu triângulo: 4
dê a medida do lado 3 do seu triângulo: 5
seu triângulo é escaleno
